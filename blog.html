1. How to Add Custom Font in CSS ( Using Downloaded File )

Here I will show you how to add custom font using download file which is eighter .eot or .woff or .otf or .svg file formate

Step 1 : Download the font
Download Font File Which You want from brower.
Step 2 : Put File
Put the file in website folder. But The file extension must be in .eot or .woff or .otf or .svg format.
Step 3 : CSS
add this css in your css file.
@font-face{
 font-family: "FontName";
 src: url("file-name.file-extension");
}
Step 4 : Declarations
Declare this font-name as you want ...
a{
  font-family: "FontName";
}
2. Move to top using Jquery

Here I will show you how to add scroll top button using Jquery, Html and Css.

Step 1 : Add Jquery File
You have to add Jquery file in head tag eighter download file or url. I will give you url link.
add this css in your head tag.
<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>

Step 2 : Add Html code in body
Put this Html Code in your body tag. I suggest put it in after <body> so you can modify at any time.
<a href="#" class="scrollToTop">Top</a>

Step 3 : Add Jquery Code
Add this jquery in your js file for or in script
$(document).ready(function(){
 $(window).scroll(function(){
  if($(this).scrollTop()>100){
   $('.scrollToTop').fadeIn();
  } else {
   $('.scrollToTop').fadeOut();
  }
 });

 $('.scrollToTop').click(function(){
  $('html, body').animate({ scrollTop: 0 });
  return false;
 });
});

Step 4 : CSS
You Can add your CSS in page as you want but I will give you a demo here.

 width: 61px;

 bottom: 25px;

 background-color: #fff;
 border: 1px solid rgb(0, 0, 0);
 overflow: hidden;
}
